So, we could use the following busybox commands to trigger CPU-GPU C2C transaction to read GPU memory data. 

sudo busybox devmem 0x200000000000 32 => to read GPU#0 memory data 

sudo busybox devmem 0x220000000000 32 => to read GPU#1 memory data 

sudo busybox devmem 0x280000000000 32 => to read GPU#2 memory data 

sudo busybox devmem 0x2a0000000000 32 => to read GPU#3 memory data
